# notes-cn-home

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件的形式存放，不依赖于特定域名或后端服务。

## 项目简介

这是一个简单的静态页面集合仓库。你可以将其视为一个轻量级的页面存档空间，用于存放、整理和分享独立 HTML 文件。所有页面均为自包含的 HTML 文档，便于本地查看或通过静态托管服务部署。

## 目录说明

```
.
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放独立 HTML 页面的目录
│   ├── example-1.html  # 示例页面一
│   └── example-2.html  # 示例页面二
└── README.md           # 本说明文件
```

- `index.html`：可选的主页文件，可提供页面索引或简单导航。
- `pages/`：主要页面存放目录。每个 HTML 文件对应一个独立的归档页面。
- 你可以根据实际需要调整目录结构，例如添加 `assets/` 文件夹存放样式或脚本资源。

## 页面归档说明

- 每个 HTML 页面应是完整的、可独立打开的文档。
- 页面内容可以是笔记、工具、演示、存档等任意静态内容。
- 建议为每个页面添加清晰的标题和元信息，方便索引和检索。
- 页面之间可通过相对路径相互链接，但不应依赖外部特定服务。

## 维护说明

- 本仓库主要维护者为仓库创建者，欢迎通过 Issue 或 Pull Request 提交改进建议。
- 新增页面时，请将其放入 `pages/` 目录，并确保文件名简洁且无特殊字符。
- 如需更新或删除页面，请直接修改对应文件或提交 PR。
- 建议保持 README 与仓库实际内容同步。
